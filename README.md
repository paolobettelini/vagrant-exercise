# Vagrant exercise

## Virtual Machine DB

Join local intent network
<br>
Install MySQL
<br>
Execute some SQL DDL
<br>
Enable remote connections

## Virtual Machine WEB

Join local intent network
<br>
Install apache2
<br>
Install PHP
<br>
Install PHP-MySQL
<br>
Forward HTTP port
<br>
Create web page to test the connection to the database

# Usage

```bash
vagrant up db
vagrant up web
```
Open browser at 127.0.0.1:9080
